# Phase 5 Working Plan

## Goal

Add building interactions on top of the current viewer: hover feedback, click or tap selection, tooltip behavior, touch-aware dismissal, and cursor updates that do not fight with map panning.

## Steps

1. Review the current viewer, entity rendering, and camera behavior relevant to Phase 5 interactions.
2. Implement hover, selection, tooltip, touch, and cursor behavior for buildings.
3. Run verification, fix any issues, and update the handoff for the next phase.

## Guardrails

- Keep the current camera and pan behavior intact.
- Avoid broad data-layer changes unless the current code blocks the interaction work.
- Treat hover and tooltip behavior as view-layer concerns.
- Make sure drag-to-pan clears hover-style UI so movement feels stable.
